# task-manager-backend
This repository contains the Node.js/Express backend for a full-stack Task Management application. It provides RESTful API endpoints for user authentication (JWT), task CRUD operations, and connects to MongoDB Atlas for persistent storage.  Tech stack: Node.js, Express, MongoDB Atlas, JWT, bcryptjs

Features:

User registration & login
JWT-based authentication
Create, read, update, delete tasks
Secure API endpoints
To run:

Install dependencies (npm install)
Configure .env with MongoDB and JWT secrets
Start server (npm start)
